
# EDUCATION_MANAGEMENT_SYSTEM

## Overview

**Education_Management_System** is a full-stack web application designed to simplify and automate the management of educational institutions. The system provides separate workflows for **Administrators**, **Trainers**, and **Students**, enabling efficient handling of academic operations.

The application follows a **scalable, modular, and secure architecture**, ensuring smooth communication between frontend and backend components.

---

## Key Features

- Role-based access for Admin, Trainer, and Student  
- Secure authentication and authorization  
- Modular and scalable backend architecture  
- RESTful APIs for frontend–backend communication  
- Angular-based frontend with modern UI  
- Easy setup and execution for beginners  

---

## System Architecture (High Level)

- **Frontend:** Angular application (Node.js & Angular CLI supported)  
- **Backend:** Java with Spring Boot (business logic & REST APIs)  
- **Database:** Managed via backend services (easily extendable)  
- **Build Tools:**  
  - Maven (Backend)  
  - npm (Frontend)  

---

## Getting Started

### Prerequisites

Make sure the following are installed on your system:

- **Java**
- **Maven**
- **Node.js**
- **npm**
- **Angular CLI**

---

## Installation

### Step 1: Clone the Repository
```bash
git clone https://github.com/SnehaDhawan/Education_Management_System
````

### Step 2: Navigate to the Project Directory

```bash
cd Education_Management_System
```

### Step 3: Install Dependencies

#### Backend (Spring Boot)

```bash
mvn install
```

#### Frontend (Angular)

```bash
npm install
```

---

## Usage

### Run Backend Server

```bash
mvn exec:java
```

### Run Frontend Server

```bash
ng serve
```

### Access Application

Open your browser and visit:

```
http://localhost:4200
```

---
